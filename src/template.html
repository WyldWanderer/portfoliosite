Face Finder- A Face Detection Web App

This web app was the culmination of the course work in a Full-Stack Development class through the Zero to Mastery Academy. Using a combination of React, Node, PostgreSQL, and API integration, the app will allow users to create a profile, sign in with that profile in the future, submit URL links to photos for face detection, and keep track of the number of submitted photos by profile.

Tweet Bot

I used to host a podcast and one of the things I always strugged with was being consistent on social media. To help with this and be able to devote more time elsewhere, I built a bot to automatically retweet the latest news from a list of sources. By using Twitter's API, Tweet Bot runs every hour gathering the latest tweet from each source, making sure it hasn't already been retweeted by the profile, then retweets anything new. The project continues to run as the podcast is ongoing and has lead to a significant increase in the rate the profile gains followers.

Background Generator

I built this tool as part of a front end development course to help with people make more dynamic backgrounds for their website and experiment with color combinations. The site will let you set your own colors and display them in a left to right gradient, or you can use the randomize button to let the site choose for you. Additionally, the code to add in the combination you create is displayed for you to use on your own website.
